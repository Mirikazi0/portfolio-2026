# Project Folders

This directory contains the standard project folder structure:

- **01_Brief** → project instructions, client notes, goals
- **02_Research** → references, inspiration, competitor stuff
- **03_Content** → text, captions, copy, descriptions
- **04_Assets** → photos, logos, icons, raw files
- **05_Design** → your working design files
- **06_Exports** → JPGs, PNGs, PDFs you export while working
- **07_Final** → polished final versions for submission or portfolio use
